Git y GitHub

Aprende Git y GitHub de forma completa y desde cero. Con ejemplos prácticos. Sé un profesional del control de versiones.

Descripción del curso

En este curso aprenderás a utilizar dos herramientas muy populares en desarrollo colaborativo de los últimos años de gran ayuda para cualquier programador front-end o back-end. 

Git es una herramienta para el desarrollo colaborativo con la que podrás controlar los cambios que se hacen en un proyecto de modo que puedas volver atrás en el historial si lo necesitas. 

Y como segunda parte del curso aprenderás a obtener el máximo potencial de GitHub, un sitio web que te permite almacenar en la nube tu código, compartirlo con los demás y sobre todo trabajar en equipo mediante las características que el sitio web te ofrece.

Temario.

Git - Unidad 1.

  • Sistema de control de versiones.
  • Conceptos básicos de git.
  • Instalación de git.

Git - Unidad 2.

  • Inicialización de git.
  • Clonar un repositorio existente.
  • Trabajando con los comandos básicos de git.
  • Manejo de confirmación de cambios.
  • Ver historial de los cambios.
  • Trabajando con repositorios remotos.

GIt - Unidad 3.

  • Creando ramas en git.
  • Trabajando con las ramas
  • Realización de merge entre las ramas.

GitHub - Unidad 4.

  • Creación y configuración de una cuenta en GitHub
  • Participando en proyectos.
  • Creacion de Pull Request.
  • Mantenimiento de un proyecto.
  • Compartir tu proyecto Git en GitHub
